Palindrome Algorithm Visualizers

Welcome to the Palindrome Algorithm Visualizers! This page provides links to two interactive visualizations that demonstrate different approaches to finding the longest palindromic substring in a given string.

Available Visualizers:

1. Expand Around Center Approach

This visualizer demonstrates the "expand around center" approach for finding the longest palindrome. It's an intuitive method that checks each possible center of a palindrome.

2. Manacher's Algorithm

This visualizer showcases Manacher's algorithm, an efficient linear-time approach to finding the longest palindromic substring. It uses dynamic programming to avoid redundant computations.

How to Use:

  1. Click on one of the links above to open a visualizer.
  2. Enter a string in the input field.
  3. Use the "Step" button to go through the algorithm step-by-step, or use "Play" for automatic animation.
  4. Observe how the algorithm processes the string and identifies palindromes.
  5. Use the speed control to adjust the animation speed if needed.

These visualizers are designed to help you understand how different algorithms approach the problem of finding the longest palindromic substring. Enjoy exploring!
